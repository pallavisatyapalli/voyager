# Voyager – Your Smart Travel Companion

Voyager is a travel-focused web application developed under the theme "Tourist and Travelling" to enhance the experience of global travelers. It provides essential utilities such as real-time currency conversion and language translation, helping users overcome common travel barriers.

---

## Features

### Currency Converter
- Convert between major world currencies instantly  
- Real-time exchange rates powered by reliable APIs  
- Simple and user-friendly interface for quick calculations  

### Language Translator
- Translate text between multiple languages on the go  
- Helps break language barriers while traveling  
- Useful for asking directions, reading signs, or basic communication  

---

## Use Case

Voyager is designed for tourists and travelers who need quick access to essential tools while visiting foreign countries. Whether shopping, dining, navigating, or communicating, Voyager simplifies cross-border interaction and improves travel convenience.

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript  
- APIs Used:
  - ExchangeRate API (for currency conversion)
  - Google Translate API 

---

## How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/voyager.git
cd voyager
````

### 2. Open the project

Open the project folder in VS Code or any code editor.

### 3. Run the project

Since this is a frontend project:

* Open `index.html` directly in your browser
  OR
* Use Live Server (recommended):

  * Install Live Server extension in VS Code
  * Right-click `index.html` → Select "Open with Live Server"
